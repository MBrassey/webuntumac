webuntu

Ubuntu GUI, Shell & Statistics accessed from the web browser.

Use the navigation menu appearing in the top left corner of this page. You will find that webuntu comes installed with an ubuntu host, accessable via the web-shell & web-gui. Go ahead and explore your new system, browse the web, write a bash script and install some useful software and services! The changes you make on the webuntu host will be persistant untill you remove the image or stop the Docker container. For more information, navigate to the README.

Browsers supported: Firefox 3.5+, Chrome any, Safari 3.2+, Opera 9.5+
